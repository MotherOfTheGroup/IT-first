Цель Лабораторной работы:

Научиться создавать html страницы, подключать к ним таблицу стилей.

Задачи

1Создать простейшую html страницу
2Подключить таблицу стилей
3Написать отчёт по лабораторной работе

План выполнения работы

    Создать папку с проектом, в ней создать следующие файлы и папки:
  1. файл "index.html". В этом файле будет содержаться основные данные страницы.
  2. папку "css", которая предназначена для хранения таблиц стилей.
      3. *внутри папки создать пустой файл "style.css"
  3. папку "sourse". В дальнейшем в этой папке будут расположены скрипты на javascript.
    4. После, нужно заполнить созданные файлы.

Ход выполнения работы

Заполнение "index.html"

Заполнение происходит путём совмещения задач 1 и 3 из таблицы. Таким образом необходимость придумывать актуальную тему, которой можно посвятить личную страницу отпадает. При выполнении работы используются простейшие теги html, такие как head, body, h1, h2, h3 , br, p, ol, li, table, tr, td, div. Также были использованы специальные теги meta и link.

Заполнение "css/style.css

После первичного заполнения файла "index.html" можно подключить таблицу стилей через конструкцию link rel="stylesheet" type="text/css" href="css/style.css. Данную конструкцию следует вставлять перед head. С помощью стилей было реализован фон.

Выводы

В ходе проделанной работы были получены знания о создании простейших html страниц, создание таблиц и реализация фона.
